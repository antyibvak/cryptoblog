# CRYPTO BLOG

Made a crypto Blog to help introduce people to blockchains and cryptocurrencies

## TechStack
React.js
Next.js
GraphQL

## TO RUN PROJECT

### PRE-REQUISITES
Node

### DIRECTIONS
- Clone repository
- Open in Text Editor
- Open terminal to folder directory
- Run "npm install" for dependencies
- Run "npm run dev"
